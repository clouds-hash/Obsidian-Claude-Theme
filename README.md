# Obsidian-Claude-Theme
A Claude-style theme for Obsidian
How to Use?

# Download the Claude Folder (is a css file and a json file)
# Open the Obsidian
# Open the settings
# In the left
# Find the Appearance
# On the Right
# Find the Theme
# Click the Folder Icon
# Copy My Folder Named Claude to you just opend Folder
# Back to Obsidian's Settings, Back to theme
# Choose the Claude Theme
# Finished

 # zh-CN 简体中文
# 先下载我的Claude文件

# 左边找到外观，然后在右边找主题文件夹，打开主题文件夹
# 把你刚刚下载的Claude文件夹复制进去
# 然后在主题选项中选择Claude
